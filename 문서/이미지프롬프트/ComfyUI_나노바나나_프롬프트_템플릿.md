# ComfyUI -> 나노바나나 프롬프트 템플릿

아래 블록을 그대로 복사해서 나노바나나에 붙여넣으세요.

## Prompt
```text
[STYLE]
- 2D game UI asset, clean silhouette, readable at small size
- hand-painted fantasy style, polished mobile game quality

[SUBJECT]
- {무엇을 만들지 한 줄}

[DETAILS]
- shape: {형태}
- material: {재질}
- color palette: {주요 색상 3~5개}
- mood: {분위기}
- ornament: {장식 요소}

[COMPOSITION]
- centered subject
- transparent/solid background: {투명 or 단색}
- no text, no watermark

[TECHNICAL]
- output: {해상도 예: 1024x1024}
- high contrast edges
- game-ready icon/sprite readability
```

## Negative Prompt
```text
blurry, low contrast, noisy texture, photorealistic, 3d render look, text, logo, watermark, cropped subject, extra objects, messy background, distorted anatomy
```

## 사용 규칙
- 파일명: `{대상}_이미지프롬프트_나노바나나.md`
- 한 파일에 `Prompt`/`Negative Prompt`를 반드시 분리.
- 실제 제작마다 템플릿 복사 후 대상 맞춤 값만 수정.
